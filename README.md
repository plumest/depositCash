## Set up
### Requirement
- Java version 8 or later
- Maven CLI 3.6

### Installation
```
$ git clone https://github.com/plumest/depositCash.git
$ cd depositCash
```

## Testing
Go to project's root directory then run following command
```shell script
$ mvn test
```
